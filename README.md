# Astrolabe

Visualize any GitHub user's starred repositories. Enter a username, get instant insights.

## Features

- Language distribution chart
- Star timeline visualization
- Repository health classification (active / stale / abandoned / archived)
- Search, filter, and sort starred repos
- Click any repo to jump to GitHub

## Tech Stack

React 19 + TypeScript + Vite + Tailwind CSS 4 + Recharts + Octokit

## Development

```bash
npm install
npm run dev
```

No environment variables or authentication needed. All data comes from GitHub's public API.
